Reolennoù CSS e penn kentañ an teuliad HTML

Ret e vo ouzphennañ ur rann nevez <style>...</style> e-barzh ar rann <head>...</head> e penn kentañ ar bajenn. Ar reolennoù CSS a vo lakaet etre <style> ha </style>

<!DOCTYPE html> <html> <head> <style> h1 {color: #00FF00;} p { color: #0000FF; font-style: italic; } </style> </head> <body> <h1>Un titl gwer</h1> <p>Ur pennad skrid liv glas hag italik</p> </body> </html>

Reolennoù CSS en ur fichennaoueg diavaez

An doare boutin. Dispartiet e vez ar stil eus an dañvez penn-da-benn. Aesoc'h eo neuze rannañ reolennoù CSS etre meur a bajenn pe eskemm stil etre lec'hiennoù Web disheñvel.

Ret eo krouiñ ur fichennaoueg nevez gant an "extension" .css (da skouer: "stil_va_bajenn.css") ha skrivañ tout a reolennoù CSS e-barzh.

Staget e vo ar fichennaoueg CSS (ar "folenn stil") ouzh an teuliad HTML en ur lakaat un elfenn nevez <link> e-barzh ar rann <head>...</head>

<!DOCTYPE html> <html> <head> <link rel="stylesheet" type="text/css" href="stil_va_bajenn.css"> </head> ... </html>

Dibab an elfennoù HTML da stummañ

Dre doare elfenn

h1 { color: #3888FF; font-size: 54px; }

Tout an elfennoù titl <h1>...</h1> eus ar bajenn a vo skoget gant ar reolenn-mañ.

Dre "id"

#skeud { box-shadow: 4px 4px 6px #A0A0A0; }

An elfenn HTML nemetañ gant an "attribut" id="skeud" a vo skoget (ur skeud griz a vo livet dindan).

Da skouer : <img id="skeud" src="url_ur_skeudenn">

Dre "class"

.e-kreiz { text-align: center; }

Tout an elfennoù HTML gant an "attribut" class="e-kreiz" a vo skoget (kreizenet an testennoù).

Da skouer :

<p class="e-kreiz">Un destenn kreizennet</p> <h2 class="e-kreiz">Un titl kreizennet</h2>

Unanennoù ar CSS

Mentoù

E pixels (da skouer: 5px)

E tregantad (da skouer: 50%)

E kantimetrad (da skouer: 0.5cm)

Livioù

E stumm hexadecimal, o kregiñ gant # hag gant 6 arouez da-heul (da skouer: #FFA309)

Gant ar fonksion rgb(0, 255, 0)

Livioù treuzvelus gant ar fonksion rgba(0, 255, 0, 0.5)

fichennaouegoù

Gant ar fonksion url("url_ur_fichennaoueg")
